# Shop System Automation 🛒

Автоматизация тестирования системы управления магазином с использованием **Playwright**.

## 🚀 Стек технологий
- **Framework:** [Playwright](https://playwright.dev/)
- **Language:** JavaScript
- **Pattern:** Page Object Model (POM)

## 📁 Структура проекта
- `tests/` — файлы тестов (регистрация, логин, админ-панель).
- `tests/pages/` — Page Object классы для взаимодействия со страницами.
- `playwright.config.js` — конфигурация Playwright.

## 🛠 Установка
1. Клонируйте репозиторий:
   ```bash
   git clone <url-вашего-репозитория>
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Установите браузеры Playwright:
   ```bash
   npx playwright install
   ```

## 🧪 Запуск тестов
- **Запуск всех тестов:**
  ```bash
  npx playwright test
  ```
- **Запуск в UI-режиме:**
  ```bash
  npx playwright test --ui
  ```
- **Запуск конкретного файла:**
  ```bash
  npx playwright test tests/login.spec.js
  ```

## 📋 Покрытые тест-кейсы (TC)
1. **Регистрация:** Успешная регистрация, валидация полей (email, телефон, пароль, уникальность).
2. **Авторизация:** Вход под разными ролями, обработка ошибок входа.
3. **Админ-панель:** 
   - Создание/удаление товаров.
   - Управление складами.
   - Изменение статусов заказов.
